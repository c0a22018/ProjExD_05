# pacman
## 実行環境の必要条件
* python >= 3.10a
* pygame >= 2.1

## ゲームの概要
パックマンが敵をよけながらコインを集めるアーケードゲーム
## 操作方法
* 上下左右キーでパックマンを操作
## ゲームの実装
###共通基本機能
* ゲームのエンジンに関するクラス
### 担当追加機能
* 敵を追加する機能
